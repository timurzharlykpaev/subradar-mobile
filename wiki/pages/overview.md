# Обзор проекта

## Что это

**SubRadar AI** — мобильное приложение для отслеживания подписок с AI-возможностями. Позволяет пользователям контролировать расходы на подписки, получать AI-рекомендации по оптимизации и прогнозировать будущие траты.

## Стек технологий

| Технология | Назначение |
|-----------|-----------|
| React Native | Кроссплатформенный UI |
| Expo SDK 51 | Инфраструктура, нативные модули |
| TypeScript (strict) | Типизация |
| Expo Router | Файловая маршрутизация |
| TanStack Query v5 | Серверный стейт, кеширование |
| Zustand | Локальный стейт |
| Axios | HTTP-клиент |
| react-native-reanimated | Анимации |
| @gorhom/bottom-sheet | Модальные действия |
| expo-notifications | Push-уведомления (Expo Push Token) |
| expo-image-picker | Камера/галерея |
| expo-av | Запись аудио |
| RevenueCat | In-app покупки |
| react-i18next | Интернационализация (10 локалей) |

## Платформы

- iOS (основная)
- Android

## Ключевые отличия от веб-версии

1. **AI голосовой ввод** — запись через микрофон, парсинг в подписку
2. **AI скриншот** — фото чека/скриншота, парсинг в подписку
3. **Push-уведомления** — напоминания о предстоящих платежах
4. **RevenueCat** — нативные In-App Purchases для монетизации
5. **Offline-first** — кеш подписок в AsyncStorage, работа без сети
