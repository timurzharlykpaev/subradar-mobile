## 1. АУТЕНТИФИКАЦИЯ

### 1.1 OTP (Email)
- [ ] Ввести email → получить код
- [ ] Ввести правильный код → вход успешен
- [ ] Ввести неправильный код → ошибка
- [ ] Отправить код повторно
- [ ] Новый email → создаётся аккаунт
- [ ] Существующий email → логин в аккаунт
- [ ] review@subradar.ai + код 000000 → вход

### 1.2 Google Sign-In
- [ ] Кнопка Google → экран авторизации Google
- [ ] Выбрать аккаунт → вход успешен
- [ ] Новый Google аккаунт → создаётся юзер
- [ ] Существующий Google аккаунт → логин

### 1.3 Apple Sign-In (iOS)
- [ ] Кнопка Apple → экран Apple ID
- [ ] Face ID / пароль → вход успешен
- [ ] "Скрыть email" → работает с relay-email
- [ ] Повторный вход → логин без создания дубля

### 1.4 Сессия
- [ ] Закрыть приложение → открыть → авторизация сохранена
- [ ] Убить приложение из памяти → открыть → авторизация сохранена
- [ ] Токен истёк → автоматический refresh
- [ ] Logout → токены очищены, переход на онбординг
